# HW2_SE_LobodaDmitry_BBI2506
Репозиторий домашней работы №2 студента ББИ 2506, Лободы Дмитрия Юрьевича. 

Проект реализует систему работы с рецептами и ингредиентами на языке Python.

В проекте реализованы следующие классы:

Ingredient — хранение информации об ингредиенте;
Recipe — хранение рецепта и работа с ингредиентами;
DietaryRecipe — расширение рецепта с информацией о диетическом типе;
ShoppingList — формирование списка покупок на основе рецептов.

Также проект содержит набор автоматических тестов, написанных с использованием библиотеки pytest.

Установка 

1. Клонировать репозиторий:
git clone <https://github.com/Dimas31321/HW2_SE_LobodaDmitry_BBI2506/tree/task2>
cd HW2_SE_LobodaDmitry_BBI2506

2. Создать и активировать виртуальное окружение:
python -m venv .venv
source .venv/bin/activate

3. Установить зависимости:
pip install -r requirements.txt

Эксплуатация
1. Запуск всех тестов:
python -m pytest

2. Запуск конкретного теста 
python -m pytest tests/test_ingredient.py
